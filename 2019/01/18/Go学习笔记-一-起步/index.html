前言

Go是一门静态类型的、支持并发的编译型语言，具备垃圾回收功能，语言风格与C有些许类似的地方。比如容器技术Docker、分布式服务发现Consul就是基于Go语言实现，因此，学习Go语言有利于了解Docker和Consul的底层实现机制，可以快速解决项目开发遇到的相关的问题。

安装

通过Go官网下载本机操作系统支持的安装版本，也可以尝试通过编译源码来安装，因为在家里使用的操作系统是MacOS，因而选择通过包安装工具Homebrew进行安装，执行brew install golang命令就会安装最新版的Go环境，安装成功后执行下面的命令，测试安装是否成功。
